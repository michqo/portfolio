## Running Locally

```bash
pnpm install
pnpm dev
```

Open [http://miqal.localhost:1355](http://miqal.localhost:1355) in your browser.

> **Fun fact:** Instead of the usual `localhost:3000`, this project uses [**portless**](https://github.com/michqo/portless) — a dev tool that maps a named local subdomain (`miqal.localhost`) to the Next.js server. No more forgetting which port is which across projects.
